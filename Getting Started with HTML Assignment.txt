Question 5:- What is the DOCTYPE Declaration in HTML?

Answer:- DOCTYPE Declaration in HTML is not a tag or any other syntax. It is basically used for “INFORMATION” to which browser is using. We are using Chrome Browser so how does the code find which browser is used for this HTML code? This is written in <!DOCTYPE> format.
